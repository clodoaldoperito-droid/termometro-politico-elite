# Termômetro Político "Tuned" - Operação Matrix

Este repositório contém a versão de elite do sistema de auditoria forense digital.

## 🚀 Funcionalidades
- **Estética Matrix Sniper:** Interface de alta fidelidade para máxima autoridade.
- **Lead Capture:** Registro obrigatório de E-mail e WhatsApp.
- **Freemium 1.0:** 1ª consulta gratuita para engajamento.
- **Monetização PIX:** Consultas extras por R$ 5,00.
- **Social Growth:** Desbloqueio de consultas via interação no Instagram.

## 🛠️ Stack
- HTML5 / CSS3 (Vanilla com Rich Aesthetics)
- JavaScript (Logic & Matrix Engine)
- GitHub Pages (Hosting)
- VPS Backend (Intelligence Engine via API)

## 📁 Estrutura de Arquivos
- `index.html`: Dashboard principal e Modais.
- `style.css`: Design System Matrix.
- `app.js`: Lógica de Auditoria, Contador e Pagamento.
